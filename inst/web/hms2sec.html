Convert h:m:s to Seconds

Usage

hms2sec(x)

Arguments

x
A vector of times in h:m:s.

Convert h:m:s to Seconds

Value

Returns a vector of times in seconds.

Description

Converts a vector of h:m:s to seconds.

Examples

hms2sec(c("02:00:03", "04:03:01"))



[1]  7203 14581



hms2sec(sec2hms(c(222, 1234, 55)))



[1]  222 1234   55
